# 자산 관리 앱 도움말

## 데이터 파일

앱을 시작할 때 JSON 파일을 선택하거나 새로 만들어야 합니다. 파일 핸들은 브라우저에 저장되어, 다음에 앱을 열면 **이전 파일 다시 열기** 버튼으로 같은 파일을 다시 불러올 수 있습니다. 모든 변경사항은 입력 즉시 파일에 자동 저장됩니다.

## 금액 단위

각 페이지 우측 상단의 단위 선택기(원 / 천 / 만)로 표시 단위를 바꿀 수 있습니다. 단위는 페이지별로 독립적으로 저장됩니다. 내부 저장은 항상 원 단위이며, 단위 변경은 표시와 입력에만 적용됩니다.

## 메인 페이지 (자산 현황)

- **총 자산**: 마지막으로 데이터가 입력된 월의 현금 + 투자 합계
- **총 부채**: 현재 월 기준 부채 합계
- **순 자산**: 총 자산 − 총 부채
- **가계부 / 투자** 버튼으로 각 메뉴로 이동

### 부채 입력

화면 오른쪽에 **부채 입력** 탭이 숨겨져 있습니다. 탭 위로 마우스를 올리면 패널이 펼쳐집니다.

- 월별로 펼쳐 항목을 추가하거나 금액을 입력합니다
- **+ 항목 추가** 버튼: 이름 입력 후 항목 생성
- **✕** 버튼: 해당 항목 삭제

## 가계부 메인 페이지

- **초기 현금 설정**: 연초(1월) 기준 보유 현금을 설정합니다. 이 값을 기준으로 이후 월의 현금이 자동 계산됩니다.
- **월별 통계 표**: 1~12월의 정기수입, 비정기수입, 지출 카테고리별 합계, 투자금변경금, 현금잔액을 한눈에 확인합니다.
- **추이 그래프**: 순수입, 총지출, 현금잔액의 월별 변화를 선 그래프로 확인합니다.

### 월 선택 카드

화면 오른쪽에 **월 선택 카드**가 숨겨져 있습니다. 탭 위로 마우스를 올리면 1~12월 버튼이 표시됩니다. 클릭하면 해당 월의 가계부 상세 페이지로 이동합니다.

## 가계부 상세 페이지

월별 수입과 지출을 직접 입력하는 페이지입니다.

### 편집 버튼

오른쪽 상단의 **편집** 버튼을 누르면 편집 모드가 활성화됩니다. 편집 모드에서만 항목 추가/삭제, 카테고리 추가/삭제가 가능합니다. **완료** 버튼을 누르면 편집 모드가 종료됩니다.

### 수입

- **정기수입**: 매월 반복되는 수입 (예: 월급). 항목을 추가하면 1월부터 12월까지 전체 월에 자동으로 생성됩니다. 항목 삭제 시 해당 월 이후 모든 월에서 삭제됩니다.
- **비정기수입**: 이번 달에만 해당하는 수입 (예: 보너스). 현재 월에만 추가됩니다.
- 편집 모드에서 항목 이름을 클릭하면 이름을 수정할 수 있습니다. **Enter** 또는 다른 곳 클릭 시 저장, **Escape** 시 취소됩니다.
- 정기수입 이름을 변경하면 해당 연도의 전체 월에 반영됩니다.
- 이미 존재하는 이름으로는 항목을 추가할 수 없습니다.

### 지출

- **카테고리**: 편집 모드에서 **+ 카테고리 추가** 버튼으로 새 카테고리를 만들 수 있습니다. 카테고리는 모든 월에 공통 적용됩니다.
- **항목**: 편집 모드에서 각 카테고리의 **+ 추가** 버튼으로 항목을 추가합니다.
- 카테고리 이름, 항목 이름 모두 편집 모드에서 클릭해 수정 가능합니다. 카테고리 이름 변경은 전체 월에 반영되고, 항목 이름 변경은 현재 월에만 반영됩니다.
- 이미 존재하는 이름으로는 카테고리나 항목을 추가할 수 없습니다.

## 투자 메인 페이지

- **카테고리별 요약**: 마지막으로 데이터가 입력된 월 기준으로 카테고리별 투자 금액을 표시합니다.
- **월별 통계 표**: 1~12월의 카테고리별 투자 금액 추이를 표시합니다.
- **추이 그래프**: 카테고리별 투자 금액 변화를 선 그래프로 확인합니다.
- **구성 비율**: 월을 선택하여 해당 월의 투자 카테고리별 비율을 파이차트로 확인합니다.
- **+ 카테고리**: 새 상위 투자 카테고리를 추가합니다.

### 월 선택 카드

가계부와 동일하게 오른쪽에 숨겨진 월 선택 카드로 투자 상세 페이지로 이동합니다.

## 투자 상세 페이지

월별 투자 현황을 입력하는 페이지입니다.

### 편집 버튼

가계부와 동일하게 **편집** 버튼으로 편집 모드를 활성화합니다. 편집 모드에서만 카테고리/항목 추가·삭제, 이름 수정이 가능합니다.

### 카테고리 구조

- **상위 카테고리**: 편집 모드에서 **+ 카테고리 추가** 버튼으로 추가합니다. 추가 시 해당 월부터 12월까지 반영됩니다. 삭제 시 해당 월 이후 모든 월에서 삭제됩니다.
- **하위 카테고리**: 각 상위 카테고리에서 **+ 하위 카테고리** 버튼으로 추가합니다.
- 카테고리/항목 이름은 편집 모드에서 클릭해 수정할 수 있습니다. **Enter** 또는 다른 곳 클릭 시 저장, **Escape** 시 취소됩니다.
- 이미 존재하는 이름으로는 카테고리나 항목을 추가할 수 없습니다.

### 투자 항목 입력

- **금액**: 해당 월의 현재 평가 금액을 입력합니다.
- **입금/인출**: 이번 달에 새로 넣은 금액(입금)이나 뺀 금액(인출, 음수)을 입력합니다. 모든 카테고리에 적용되며, 이 금액은 가계부의 투자금변경금에 자동 반영되어 현금에서 차감/추가됩니다.
- **손익 표시**: 전월 대비 수익/손실이 자동 계산되어 표시됩니다. (계산식: 현재금액 − 전월금액 − 입금액)

## 계산 방식

| 항목 | 계산 |
|---|---|
| 현금 (1월) | 초기현금 + 순수입 − 총지출 + 투자금변경금 |
| 현금 (2월~) | 전월현금 + 순수입 − 총지출 + 투자금변경금 |
| 투자금변경금 | 전체 투자 항목의 입금액 합계를 음수로 적용 (현금 차감) |
| 총 자산 | 마지막 활성 월의 현금 + 마지막 활성 월의 투자 합계 |
| 순 자산 | 총 자산 − 총 부채 |
