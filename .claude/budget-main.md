### 2. 가계부 메인페이지

- 최상단에는 마지막으로 계산된 "순수입", "총지출", "현금" 금액을 보여준다. 기본은 현재 달의 계산된 금액이겠지만, 이번달의 금액을 입력하지 않았다면 이전에 마지막으로 입력했던 달의 금액을 노출하면 된다.
- 우측 상단에 초기 현금을 입력할 수 있는 버튼을 둔다. 초기 자산에는 현재의 "현금"을 둔다. 매년 1월의 현금 잔액 계산은 초기 자산의 "현금" - 1월의 "순수입"-"총지출"+"투자금변경금"이 될 것이다.
- 1월을 제외한 달의 계산은 직전월의 현금에서 해당월의 순수입-총지출이 될 것이다.
- 월별 통계를 보여주도록 표로 구성한다. 표는 1~12월로 구분되는 각 Column이 존재하고 "정기수입", "비정기수입" "각 지출 카테고리", "투자금 변경"을 row로 구성한다.
- 각 row의 항목은 합산 금액만 표시하면 된다. 예를 들어, 정기수입의 항목이 5개라면, 해당 항목의 합산 금액만 통계표에 구성하면 된다.
- 각 월별 통계 표를 기반으로 그래프를 그린다. 월별 금액을 그래프로 그려 사용자가 추이를 파악할 수 있도록 한다.
- 오른쪽에 hover하면 card가 밀려서 나오는 기능을 구현한다. card에는 세로로 1월~12월이 작성되어 있다. 특정 월을 선택하면 해당 월의 상세 가계부 페이지로 이동할 수 있다.(4)
- 단위 관리가 되어야 함
